GitおよびGitHubの使い方をメモしておくためのサイトです。(自分の備忘録)
コマンド
コマンド説明備考
git initリポジトリの初期化新規作成時
git clone <github URL> [作成するディレクトリ名]リポジトリの複製既存リポジトリの取得時
git add <ファイル名>ファイルの追加変更ファイルのステージング
git resetステージングのキャンセル 
git commit [-m "コミットメッセージ"]変更のコミットステージングしたファイルのコミット
git logコミット履歴の確認 
git log --onelineコミット履歴の簡易表示 
git branch作業ブランチの確認 
git branch <ブランチ名>新規ブランチの作成 
git branch -d <ブランチ名>ブランチの削除 
git checkout <ブランチ名>作業ブランチの切り替え 
git push <リモートリポジトリ名> <ローカルリポジトリ名>リモート(github)へのプッシュローカルの変更をリモートに反映
git pull <リモートリポジトリ名> <ローカルリポジトリ名>リモート(github)からのプルリモートの変更をローカルに反映
git statusステータスの確認ファイル変更の有無を確認
git diff [ファイル名]変更箇所の確認ファイル内変更箇所の確認
フォークリポジトリの複製他のユーザーのリポジトリを複製
新規リモートリポジトリの登録
  1. GitHubにログイン
  2. Repositoriesタブをクリック
  3. Newボタンをクリック
  4. Repository nameにリポジトリ名を入力
  5. Public or Privateを選択
  6. Create repositoryボタンをクリック
  7. Quick setup にて新規か既存リポジトリの連携かを選択し、コマンドラインをコピーして実行
    (補足)README.md をCreate repositoryボタンをクリック時に行うとこの手順は出ない
